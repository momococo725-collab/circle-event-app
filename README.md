# Campus Event Manager

大学サークルや学生団体向けのイベント管理アプリです。  
イベントの登録・参加管理・削除をブラウザから簡単に行えます。

## URL
https://circle-event-app.vercel.app

## 機能
- イベント追加
- 参加 / 未参加切り替え
- イベント削除
- カレンダー表示

## 使用技術
- Next.js
- React
- TypeScript
- Tailwind CSS
- Git / GitHub
- Vercel（デプロイ）

## 作成背景
大学のサークルや学生団体では、  
イベント管理や参加確認がLINEやスプレッドシートで分散していることが多いと感じました。  
そこで、イベント管理を一つの画面で完結できるWebアプリを制作しました。
